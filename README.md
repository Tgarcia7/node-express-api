# NODE-EXPRESS-API

RESTful API written in Node.js and Express framework. It uses JWT for authentication.

### Initial config 

```
DB_URI=mongodb://mongo:27017/
DB_NAME=node-express-api
```

### Run

```bash
$ docker-compose up -d
```

### Lint

```bash
$ docker-compose run --rm api bin/lint
```
